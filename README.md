**monorepo-react-and-native**

Template for Mobile/Web app that utilizes code sharing. 

Hint: Start mobile server before web server for concurrent testing

|Command|Description  |
|--|--|
| yarn install | Installs dependencies |
| yarn compile | Compile typescript files |
| yarn workspaces info | Workspace information |
| yarn workspace web start | Run HTTP server* |
| yarn workspace mobile android | Run Android server* |
| yarn workspace mobile ios | Run iOS server* |
